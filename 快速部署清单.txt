🚀 快速安装步骤（5步）
=====================================

第1步: 解压文件
-------------------------------------
将 PO-workflow-package.zip 解压到任意目录
例如: D:\PO-workflow


第2步: 运行安装脚本
-------------------------------------
双击运行: quick-start.bat
或在 PowerShell 中: .\quick-start.ps1

脚本会自动:
  ✓ 检查 Node.js 环境
  ✓ 安装前后端依赖
  ✓ 安装 Chromium 浏览器
  ✓ 启动服务


第3步: 配置环境变量（首次运行必需）
-------------------------------------
编辑文件: server\.env

填写以下信息:
  OPENAI_API_KEY=你的API密钥
  PROCORE_EMAIL=你的邮箱
  PROCORE_PASSWORD=你的密码


第4步: 访问应用
-------------------------------------
浏览器打开: http://localhost:3000


第5步: 在Web界面测试配置
-------------------------------------
1. 点击右上角"设置"
2. 点击"Test API Key"测试
3. 保存设置

=====================================
⚠️ 重要提示
=====================================

1. 首次安装需要下载依赖（约 200-300MB）
   请确保网络连接稳定

2. API Key 安全
   - 不要分享你的 .env 文件
   - 不要将 API Key 提交到 Git

3. 端口占用
   - 前端: 3000
   - 后端: 3001
   如有冲突，请先关闭占用端口的程序

4. Playwright Chromium
   首次使用自动化功能时会自动下载
   大小约 240MB
